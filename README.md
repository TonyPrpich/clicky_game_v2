React Memory Game:
Simple React.Js memory game, picture changes location on every click, User tries to get to 12 points without clicking the same picture twice!

Getting Started-

Dependencies:
React.Js

Installing:
npm install

Executing program:
npm start

Technology Used:
JavaScript, React.js, CSS, HTML5
